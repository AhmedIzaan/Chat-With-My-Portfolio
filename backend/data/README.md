# Resume Placeholder

Please replace this file with your actual resume (PDF or DOCX format).

**Instructions:**

1. Remove this README.md file
2. Place your resume here as `resume.pdf` or `resume.docx`
3. Commit the resume file to your repository

**Supported formats:**
- PDF (.pdf)
- Microsoft Word (.docx)

**Note:** The resume file MUST be committed to the repository for the system to work on Render free tier, as there is no persistent disk storage.

**File path should be ONE of:**
- `backend/data/resume.pdf`
- `backend/data/resume.docx`

The system will automatically parse and load your resume into the vector database on startup.
